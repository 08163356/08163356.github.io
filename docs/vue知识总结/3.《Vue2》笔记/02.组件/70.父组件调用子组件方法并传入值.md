# 父组件调用子组件方法并传入值

通过`ref`引用调用子组件内的方法并传入参数

父组件：

```js
<子组件标签  ref="refName"></子组件标签>

methods: {
    fnX(x) {
      this.$refs.refName.fnY(x) // 调用子组件方法并传入值
    }
}
```

子组件：

```js
methods: {
    fnY(x) {
      this.x = x
    }
  }
}
```
